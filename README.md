## Проект
### Тесты
Тестируются ручки. На корректность работы и выдачи нужных ошибок.

- Проверяет создание заказа с авторизацией, создание заказа без авторизации, создание заказа с ингредиентами, заказ без ингредиентов, заказ с неверным хешем (`test_order.py`)
- Проверяет создание уникального пользователя, дубликат пользователя, создать пользователя без "email", логин с существующим пользователем, логин с неверными данными (`test_user`)


## Запустить тесты 
pytest -v 
## Установить зависимости 
pip install -r requirements.txt
